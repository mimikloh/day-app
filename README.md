# Day App

**Day** — это минималистичное приложение для планирования дня, привычек и задач с красивым интерфейсом, тёмной/светлой темой, синхронизацией через Firebase и поддержкой категорий.
Приложение помогает организовать расписание, отслеживать привычки и видеть прогресс в виде красивого кольца или линейного индикатора активности.

## Основные возможности

- Планирование задач на день с временем и датой
- Вневременные привычки (habits) без привязки к расписанию
- Категории/тематики с иконками и цветами (можно создавать свои)
- Красивое кольцо активности или линейный прогресс (на выбор)
- Тёмная и светлая темы + режим «как в системе»
- Авторизация через email/password + Firebase
- Облачная синхронизация задач, привычек и категорий
- Умное удаление категорий
- Принудительное удаление с подтверждением (удаляет связанные задачи и привычки)
- Push-уведомления (базовая настройка)

## Технологии

- Flutter (Dart)
- Riverpod (управление состоянием)
- Firebase (Authentication + Realtime Database)
- Hive (локальное хранение)
- flutter_slidable (свайп для удаления)
- go_router (навигация)
- shared_preferences (сохранение настроек)
